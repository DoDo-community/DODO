## What is DODO?

두두는 **"해야지 해야지 하면서 미루던 것"** 들을 다같이 모여서 밤새 해치우는 일입니다. <br>
혼자 하는 것 보다 모여서 하는 것이 집중력에 더 도움이 되며, <br>

각 분야의 여러사람들을 만나 서로 이야기도 나눠보아요.
